# 📱 PokéFlow — Explorador de Pokémon

O **PokéFlow** é uma aplicação web responsiva desenvolvida para facilitar a consulta de dados sobre Pokémon. O projeto utiliza a **PokeAPI** para buscar informações em tempo real e apresenta uma interface dinâmica que se adapta visualmente ao tipo do Pokémon pesquisado.

> **Status do Projeto:** Concluído ✅

## 🚀 Funcionalidades

* **Busca em Tempo Real:** Pesquisa por nome ou ID oficial.
* **Design Mobile-First:** Interface otimizada prioritariamente para smartphones.
* **Cores Dinâmicas:** O fundo do card muda de cor automaticamente de acordo com o tipo principal do Pokémon (ex: Vermelho para Fogo, Azul para Água).
* **Tratamento de Erros:** Alertas visuais caso o Pokémon não seja encontrado ou haja erro de conexão.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando o "trio fundamental" do desenvolvimento Web, sem frameworks externos, para demonstrar domínio sobre a linguagem base:

1.  **HTML5**: Estrutura semântica da página.
2.  **CSS3**: Estilização moderna com Flexbox e Media Queries para responsividade.
3.  **JavaScript (ES6+)**:
    * `Fetch API` para consumo de dados assíncronos.
    * `Async/Await` para manipulação de promessas.
    * Manipulação dinâmica de DOM.
